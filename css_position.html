Position คือ การจัดตำแหน่งของวัตถุ
  • Static - จัดวางแบบปกติเป็นค่า default
  • relative - ใช้ในการจัดวางและกำหนดตำแหน่งวัตถุโดยการจับจากจุดที่วัตถุนั้นๆอยู่
  • adsolute - ใช้ในการจัวางวัตถุ ให้ไปอยู่ตำแหน่งใดก็ได้แต่ต้องกำหนดตำแหน่งจากขอบของ element ที่บรรจุวัตถุนั้นๆอีกที (ถ้าไม่มีอะไรครอบก็ถือว่า body เป็น element ที่ครอบวัตถุที่ระบุ adsolute)
  • fixed - ใช้ในการจัดวางวัตถุให้อยู่ตำแหน่งเดิม
  • sticky - ให้วัตุติดขอบเมือเลื่อนไปถึง
