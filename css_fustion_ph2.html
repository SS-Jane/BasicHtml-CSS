Flex Item

การกำหนด Propertise ให้กับ Item ใน Flexbox

Flex-basic กำหนดค่าความยืดหยุ่นเริ่มต้น

Flex-shrink หดเป็นกี่เท่าถ้าเทียบกับ item อื่น

Flex-grow ขยายเป็นกี่เท่าถ้าเทียบกับ Item อื่น

Flex Justify

การจัดวางตำแหน่งของ Item เทียบกับแกนหลัก ของ Container

CSS variable

การกำหนดตัวแปรเอง (custom variable) โดยขึ้นต้นด้วยเครื่องหมาย - แลัวตามด้วยชื่อตัวแปร

โดยใช้ :root {ตัวแปรทั้ต้องการกำหนด} และจะเลือกใช้โดย propoties : var(ตัวแปร)

                    :root{
                        --spacing: 1%;
                    }
                    .header-1{
                        magin:var(--spacing);
                        padding: var(--spacing);
                    }
